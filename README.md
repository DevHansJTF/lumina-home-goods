# Lumina Furniture Services

A premium furniture showcase and inquiry-based e-commerce platform built with modern frontend technologies, focused on delivering immersive UI/UX, responsive layouts, and refined user interactions inspired by luxury digital storefronts.

---

## Overview

Lumina Furniture Services is a modern web platform designed to showcase curated furniture and interior products through a visually polished and interactive browsing experience. The project emphasizes minimalist design, smooth animations, structured navigation, and responsive user interfaces to create a sophisticated digital showroom experience.

Users can explore categorized furniture collections, browse product details, and submit inquiries through a streamlined inquiry-based checkout workflow.

---

## Features

* Interactive product galleries with filtering and categorization
* Inquiry-based checkout workflow
* Responsive design optimized for desktop and mobile devices
* Smooth scrolling and transition-based interactions
* Modern minimalist UI/UX design
* Reusable and scalable frontend component architecture
* Dark and light mode support
